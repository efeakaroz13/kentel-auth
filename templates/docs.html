kentel-auth Official Documentation


Setup

First clone our repository with this command:

git clone https://github.com/efeakaroz13/kentel-auth

Then you need to setup the requirements and the C program which is the core of module.

python3 -m pip install -r requirements.txt
gcc -o auth-module auth.c

You are ready to go!



Usage

Python usage

Note: all functions returns json: {"SCC":Bool}

First of all define your project name:

myproject = kauth("myproject")

Register

myproject.createuser(username="",password="",fullname=None,city=None,birthyear=None,talents=None)

Explanation of this: username and password are required paramaters for using .createuser function but other ones are not required they are optional.



Login

myproject.login(username="",password="")

You can use this function with just your username and password!



User Information(JSON)

myproject.getuser(username)

This function displays all information about the user.



Change Password

myproject.changepassword(username,oldpassword,newpassword)

You need to specify the old password while using this function. it basically changes your password with python(auth-module can't handle it.)



Removing user

myproject.remove(username,password)

You need to enter username and password for removing the user from system. It completely deletes user from system
